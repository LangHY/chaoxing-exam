## 关键发现

### 1. 填空题章节可能没有重做按钮
- 章节 4.5（10单选+3填空）：提交后状态为"待批阅"，无重做按钮
- 原因：填空题需要人工批改，系统不允许重做直到批改完成
- 影响：纯单选题章节可重做满分，含填空题的章节可能无法重做

### 2. 重做后 JS 加载问题
- 章节 10.5：重做后 `btnBlueSubmit` 和 `UE` 均为 `undefined`
- frame URL 带 `reEdit=2` 参数
- 选择题可填写（纯 DOM click），但无法提交（`btnBlueSubmit` 不可用）
- 此问题不总出现，章节 7.8、8.7、9.5、11.4 的重做都正常

### 3. 视觉模型识别准确率
- 文字题目：95%+ 准确
- 数学公式（LaTeX）：90%+ 准确，偶有符号错误
- 图表/波形图：70-80%（需要 Agent 推理补充）

### 4. 最佳实践
- 先做纯单选题章节（重做成功率高）
- 含填空题的章节放在最后（可能无法重做）
- 每章结束后检查成绩，有错题立即重做（不要等到最后）
