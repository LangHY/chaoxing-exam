- **失败原因**：1章无重做按钮（待批阅），1章重做后 JS 未加载（reEdit=2）

## 关键发现

1. **首次正确率与题目类型强相关**：纯概念题 >80%，计算题 50-70%
2. **重做机制是核心优势**：即使首次答错，系统会显示标准答案，重做时直接填正确答案
3. **reEdit=2 问题是最大障碍**：重做后 frame JS 不加载导致提交失败，目前无可靠解决方案
4. **填空题/简答题导致"待批阅"状态**：这些章节可能没有重做按钮
5. **10-15 题的章节耗时约 2-3 分钟**（含视觉提取 + 填写 + 提交）
